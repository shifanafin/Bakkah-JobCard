# Bakkah Website UI Redesign — Sketch/Blueprint Aesthetic

## Goal
Transform the Bakkah Premium Auto Care website from its current modern dark/gold style into a **hand-drawn sketchbook/blueprint aesthetic** — like a luxury automotive workshop sketchbook on cream parchment paper, with pencil-drawn illustrations, handwritten labels, and mechanical motifs.

## Reference Design
The user provided a mood board showing:
1. **Hero Section** — Hand-drawn luxury workshop scene (porsche in bay, detailers at work), labeled "BAKKAH AUTOS", with pencil-sketch style
2. **UI Design Elements** — Sketch-style gauges, switches, metal badges, car keys as decorative UI elements
3. **Service Section** — Hand-drawn cards for Car Detailing, Ceramic Coating, PPF, Mechanical, Recovery, Fleet
4. **About Section** — Isometric workshop layout sketch (reception, inspection bay, service bays, quality check, delivery area)
5. **Gallery** — Polaroid-style photos pinned to a corkboard/sketchbook
6. **Testimonials** — Handwritten-style review cards with customer names and ratings
7. **Animations** — Sketch reveal effects

## Existing Foundation (Reuse)
The project already has significant sketch/blueprint infrastructure:
- `components/marketing/SketchbookBackground.tsx` — full decorative backdrop with grid, ruler ticks, gear motif, tire tread, grain texture
- Color tokens: `paper` (cream), `graphite` (charcoal), `blueprint` (navy), `silver`
- Font: `font-hand` (Caveat) already loaded for handwritten text
- `font-display` (Bebas Neue) for headings
- Grid patterns (`grid-pattern`, `grid-pattern-dark`) in Tailwind config

## Implementation Plan

### Phase 1: Theme & Global Styles
**Files:** `tailwind.config.ts`, `app/globals.css`, `app/layout.tsx`

1. **Default to light mode** — Change `<html>` default class from `"dark"` to `"light"` so the sketchbook/paper aesthetic is the default experience
2. **Add sketch-specific Tailwind utilities:**
   - `sketch-border` — dashed border that mimics hand-drawn lines
   - `sketch-card` — card with paper texture, pencil border, slight rotation
   - `pencil-text` — text with slight roughness filter
3. **Add CSS filter for sketch effect** — SVG filter (`feTurbulence` + `feDisplacementMap`) for hand-drawn line distortion on borders
4. **Add paper texture background** — subtle repeating noise pattern for the cream paper feel
5. **New keyframe animations:**
   - `sketch-draw` — stroke-dashoffset animation for SVG path drawing effect
   - `pencil-reveal` — clip-path or mask animation revealing content like pencil shading
   - `pin-drop` — for gallery polaroid pins

### Phase 2: Hero Section Redesign
**Files:** `app/HomeClient.tsx` (hero section), new `components/hero/SketchHeroIllustration.tsx`

1. **Replace the current hero image placeholder** with a full-width hand-drawn SVG workshop illustration
   - Create `components/hero/SketchHeroIllustration.tsx` — an inline SVG component depicting:
     - Workshop bay with a luxury car (Porsche silhouette)
     - Detailers at work
     - Labeled callouts: "Mechanic Bay", "Detailing Station", "Quality Check Station"
     - "BAKKAH AUTOS" as a sketch-style badge/sign
     - Hand-drawn style with `stroke-dasharray` animation for the draw-in effect
   - Use `font-hand` (Caveat) for all labels
   - Graphite stroke color on paper background
2. **Update hero text area:**
   - Keep the existing headline structure but change background from dark to paper-100
   - Add handwritten subtitle with `font-hand`
   - Add sketch-style corner brackets (already partially exist)
   - Add blueprint grid pattern behind the text
3. **Replace hero CTA buttons** with sketch-style buttons:
   - Primary: golden gradient with pencil-style border (slightly irregular)
   - Secondary: dashed border, paper background
4. **Add decorative sketch elements around hero:**
   - Small gear SVGs, wrench SVGs floating as subtle decorations
   - Ruler/measurement marks along edges
   - "Fig. 01" style handwritten labels

## Notes
- The existing `SketchbookBackground.tsx` and color tokens (`paper`, `graphite`, `blueprint`, `silver`) are excellent foundations — we build on top of them
- `font-hand` (Caveat) is already loaded — use it extensively for labels, captions, handwritten text
- `font-display` (Bebas Neue) stays for section headings — gives contrast with the handwritten elements
- All SVG illustrations will be inline components for easy animation with Framer Motion
- Keep the existing CMS-driven image system (services, gallery, before/after) — just change the framing/styling
- The WhatsApp/CTA functionality stays the same — just visual wrapper changes
